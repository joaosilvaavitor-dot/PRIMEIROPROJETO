 Primeiro Projeto – Calculo de Consumo de Energia

 

Este projeto em Python foi criado para Calcular o consumo de equipamento eletroeletrônicos

A fómula utilizada foi A potência, multiplicada pela quantidade de tempo usado por dia, multiplicado pela quantidade de dias no mês, dividido por 1000 por conta que 1kW equivale a 1000W de potência, basicamente unidades de medida.

Basicamente o programa pede informações de potência do equipamento e quantidade de tempo que será utilizado por dia, para poder calcular o consumo.
